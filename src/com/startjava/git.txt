git init добавление папки в гит
git rm удаление файла 
git status статус
git add добавление в область индекса
git commit -m создание киммита
git commit -a -m " " создание коммита без добавления индекса
git mv 1название 2 название  переименовать фаил
git log --oneline  краткая история изменений 
git log полная история изменений
 .git ignore. для того, чтобы игнорировать некоторые файлы создать текстовый документ с эти именем и добавить туда название игнорируемых файлов
 git checkout master переключение между репозиториями
